# Mimir Operators

This repository contains the Juju charms for [Grafana Mimir](https://grafana.com/oss/mimir/), following the [coordinated-workers](https://github.com/canonical/cos-coordinated-workers/) pattern:

- **coordinator** (`coordinator/`): The Mimir coordinator charm, previously at [canonical/mimir-coordinator-k8s-operator](https://github.com/canonical/mimir-coordinator-k8s-operator).
- **worker** (`worker/`): The Mimir worker charm, previously at [canonical/mimir-worker-k8s-operator](https://github.com/canonical/mimir-worker-k8s-operator).
